# 算法部分

前端面试算法肯定会考，但是相对服务端开发或者专门的算法工程师来说肯定会问的比较简单基础一些
需要了解基本的数据结构和数据结构之间的转换，比如用两个栈去实现一个队列等
基本上在 LeetCode 上刷一些简单和中等难度的题就可以 不过切记要熟练掌握，因为面试官有可能不会考原题，而是基于 LeetCode 的题进行稍微的改动，要根据自己的理解灵活的去处理，不要死记硬背某种算法的答案

# 三大框架 MVVM

框架的话基本不会三个都问，说自己最了解、擅长的。
比如 vue 的话，最好对源码有一定程度的理解，不要面试的时候只能说出双向绑定和使用 proxy 等，这种答案基本每个面试者都能说得出来，不能真正打动面试官
可以进一步深入一些，比如 vue 的 diff 算法等
